Modelo de Base de Datos — iBichos

Base de datos: Cloud Firestore (NoSQL)
Colecciones: users, captures

✅ Requerido — el usuario lo ingresa  |  🔄 Auto — gestionado por el sistema  |  ⚪ Opcional — puede estar vacío o nulo

Colección: users

Documento ID: uid del usuario (Firebase Auth)

Campo Tipo Requerido Descripción
uidString✅ RequeridoID único del usuario (Firebase Auth)
displayNameString✅ RequeridoNombre del usuario
emailString✅ RequeridoCorreo electrónico
regionString✅ RequeridoRegión del usuario
comunaString✅ RequeridoComuna del usuario
birthDateString✅ RequeridoFecha de nacimiento
genderString✅ RequeridoGénero
avatarUrlString?⚪ OpcionalURL de la foto de perfil
xpLong🔄 AutoXP acumulado (inicia en 0)
levelString🔄 AutoNivel actual (inicia en "Casual")
totalCapturesInt🔄 AutoTotal de capturas (inicia en 0)
uniqueInsectsCountInt🔄 AutoEspecies únicas capturadas (inicia en 0)
medalsList<String>🔄 AutoLista de medallas obtenidas (inicia vacío)
medalsEarnedAtMap<String, Long>🔄 AutoTimestamp (epoch ms) por medalla ganada
levelUpAtMap<String, Long>🔄 AutoTimestamp (epoch ms) por nivel alcanzado
categoryCountsMap<String, Int>🔄 AutoCapturas por categoría taxonómica

Campo Tipo Requerido Descripción

imageUrlString✅ RequeridoURL de la imagen del insecto
insectNameString✅ RequeridoNombre común del insecto (Kindwise)

dangerLevelString🔄 AutoNivel de peligro (default: "Inofensivo")

xpAwardedLong🔄 AutoXP otorgado por la captura (default: 50)
latitudeDouble?⚪ OpcionalCoordenada de latitud GPS
longitudeDouble?⚪ OpcionalCoordenada de longitud GPS
descriptionString⚪ OpcionalDescripción del insecto
